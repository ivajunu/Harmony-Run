---
    title: Getting a recommended workout from daily health component
    ---

    sequenceDiagram

    actor U as User
    participant F as Frontend
    participant E as Backend
    participant P as Database

    U->>F: Fills in daily health form
    F->>E: Gets the answer and sends the answer to backend
    E->>P: Fetches data from the right table with the answer
    P-->>E: Sends back the requested table
    E-->>F: Replies with the same table as JSON
    F-->>U: Frontend randomizes and chooses one recommended workout
